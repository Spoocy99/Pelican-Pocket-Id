# Pelican PocketId OIDC Provider Plugin

Plugin for the [Pelican Panel](https://pelican.dev) that adds PocketId as an OIDC authentication provider.

## How to install
* Download the repository archive
* Extract the `pocketid-oidc-provider` folder and place into the `plugins` folder (`/var/www/pelican/plugins` by default). 
* Open your panel and head to the plugins page and click on "Install".
